Recherche
Avancée

Chatbot IA
Qualifiant

Capture et qualifie vos leads 24h/24.

  • 1
    Le bouton flottant apparaît sur toutes les pages du site agence.
  • 2
    L'IA engage la conversation en détectant automatiquement la langue du visiteur (fr, en, es, nl, pl, ar…).
  • 3
    Elle qualifie le projet : budget, localisation, délai, type de bien, puis demande les coordonnées.
  • 4
    Le lead est enregistré en base Supabase et apparaît instantanément dans le Mini CRM.
🌍
13 langues supportées — détection automatique. Le chatbot répond toujours dans la langue du visiteur.

🏠 Module 3 / 4

Mon Espace
Property Manager

Gérez votre catalogue de biens directement depuis le site.

  • 1
    Activez le module depuis le dashboard admin — le lien "Mon espace" apparaît dans le footer.
  • 2
    Première connexion : créez votre mot de passe (min. 8 caractères, stocké en hash bcrypt).
  • 3
    Ajoutez, modifiez, supprimez vos biens avec photos, description, prix et caractéristiques.
  • 4
    Les biens s'affichent immédiatement sur le site public après enregistrement.
🔐
Sécurité : session de 8h en sessionStorage, mot de passe bcrypt, API protégée côté serveur (service role key).

Mini CRM
Leads

Suivez et gérez vos prospects en temps réel.

  • 1
    Activez le module depuis le dashboard admin — lien "Mes leads" dans le footer.
  • 2
    Créez votre mot de passe CRM à la première connexion. Indépendant de Mon Espace.
  • 3
    Visualisez tous les leads capturés par le chatbot : nom, email, téléphone, budget, localisation.
  • 4
    Faites évoluer le statut d'un clic : Nouveau → Contacté → Converti.
